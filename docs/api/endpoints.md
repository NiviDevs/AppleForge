# API Endpoints

## GET /health
Returns service health.

## POST /score
Input:
```json
{
  "spo2": 96,
  "hr": 78,
  "bp": 120,
  "roomClass": "ER",
  "equipmentTier": "T2",
  "energy": {
    "pgrid": 1,
    "batteryLevel": 80,
    "fuel": 70
  }
}
```

Response includes score breakdown, policy decision, and decision hash.

## GET /stream
SSE stream of decisions generated by simulator ticks.

## POST /scenario
Body:
```json
{ "scenario": "normal" }
```
Allowed values: `normal`, `outage`, `critical`.
